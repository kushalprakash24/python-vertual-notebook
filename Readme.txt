# Virtual Notebook

A simple, interactive virtual notebook application built with Python. This project allows users to take and save notes, search through their notes, and organize them efficiently.

## Features

- Create Notes: Users can add new notes with titles and content.
- Search Notes: Search for specific notes based on keywords.
- Save and Edit: Notes are saved persistently and can be edited later.
- Delete Notes: Option to delete unnecessary or outdated notes.
- Categorization: Organize notes into categories for better management.
